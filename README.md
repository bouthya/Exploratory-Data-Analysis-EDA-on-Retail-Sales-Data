# Exploratory Data Analysis (EDA) on Retail Sales Data

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on retail sales data to uncover meaningful business insights. The analysis helps understand customer purchasing behavior, product performance, and sales trends over time.

## Objectives

* Load and clean retail sales data.
* Perform descriptive statistical analysis.
* Analyze sales trends over time.
* Examine customer demographics and purchasing behavior.
* Identify top-performing product categories.
* Create visualizations to communicate insights effectively.
* Provide business recommendations based on findings.

## Dataset Information

The dataset contains retail transaction records including:

* Transaction ID
* Customer ID
* Product Category
* Quantity Purchased
* Price Per Unit
* Total Amount
* Customer Age
* Customer Gender
* Purchase Date
